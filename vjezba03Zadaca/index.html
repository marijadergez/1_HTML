Osnove CSS rasporeda

Tri ključne tehnike: Flexbox, Grid i Float

Flexbox Grid Float

1. Fleksibilni model kutija (Flexbox)

Flexbox je jednodimenzionalni model rasporeda dizajniran za distribuciju prostora između stavki u spremniku i moćnu mogućnost poravnanja. Idealan za navigaciju ,male komponente i centriranje elemenata.

Ovdje je prikazano poravnanje i automatska prilagodba stavki unutar reda.

2. Dvodiomenzionalni model mreže (css Grid)

CSS Grid je dvodiomenzionalan sustav rasporeda (redobvi i stupci). Savršen je za dizajniranje cjepokupnog rasporeda stranice (main lyout) gdje želite precizno pozicioniranje elemenata.

Elementi su smješteni pomoću svojstva grid template areas.

3. Klasični rasporedi (Float Columns)

Prije Flexboxa i Grida koristio se Float za bočno pozicioniranje elemenata. **Multi-column** omogućuje pakiranje sadržaja u stupce.

Dodatne informacije

Sva tri modela su i dalje relevantna.Flexbox za komponente,Grid za cjelokupni raspored stranice. a Columns za tekstualne blokove.

Primjer kreiran za demonstraciju CSS layout svojstava.
